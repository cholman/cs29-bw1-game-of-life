Conway's Game of Life

The Rules

  • For a space that is 'populated':
  • Each cell with one or no neighbors dies, as if by solitude.
  • Each cell with four or more neighbors dies, as if by overpopulation.
  • Each cell with two or three neighbors survives.
  • For a space that is 'empty' or 'unpopulated'
  • Each cell with three neighbors becomes populated.

About Conway's Game of Life

John Horton Conway devised this cellular automaton in 1970. It is a zero player game, determined by it's initial state. The Game of Life is said to be Turing complete.

Turing Complete means that it must be able to do everything a Turing machine can do. The ability to read and write ones and zeros to an infinite length of tape is enough to compute anything.
